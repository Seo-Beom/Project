## 💼 주제 D. 채용박람회·커리어 이벤트 관리 플랫폼 (Career Connect)

- **특징:** 기업 부스 관리 및 상담 예약 중심의 채용 이벤트 관리 및 이력서 제출 처리
- **역할 분담:**
  - **팀원 1:** 회원 인증(JWT), 역할별 가입(구직자/기업/운영자), 마이페이지, 프로필 관리
  - **팀원 2:** 박람회 이벤트 CRUD, 기업 부스 등록/관리, 채용 공고 등록, 카테고리 필터링
  - **팀원 3:** 상담 예약 시스템(시간대 슬롯), 예약 충돌 방지 로직, 캘린더 UI(react-big-calendar)
  - **팀원 4:** 이력서 업로드/관리(파일 어댑터), 기업측 이력서 열람, 관심 기업 북마크
  - **팀원 5:** 참가/상담 통계 대시보드(Chart.js), 관리자 승인 페이지, 이벤트 전체 관리

---

### 1. 채용박람회의 세부 의미

- **채용박람회(Job Fair):** 여러 기업이 한 장소에 부스를 차리고, 구직자들이 방문하여 기업 정보를 얻고 이력서를 제출하는 대규모 채용 이벤트. 오프라인 + 온라인 병행이 최근 추세.
- **채용설명회(Recruitment Session):** 한 기업이 자사 소개, 직무 소개, 채용 절차 등을 설명하는 세미나형 이벤트. 박람회 내에서 부스별 미니 세션으로 진행되기도 함.
- **기업 부스(Booth):** 박람회 내 각 기업에 배정된 공간. 기업 소개, 채용 공고, 상담이 이루어지는 단위. 부스 번호, 위치(구역/층) 정보가 포함됨.
- **1:1 상담 예약:** 구직자가 관심 기업에 사전 상담 예약을 하는 시스템. 시간대별 슬롯(15~30분 단위)을 선택하며, 한 슬롯에 한 명만 예약 가능 (동시성 제어 필요).
- **이력서 제출(Resume Submission):** 구직자가 관심 기업에 이력서를 온라인으로 제출. 기업 HR 담당자는 접수된 이력서를 열람·관리.
- **부스 배치도(Booth Map):** 박람회 전체 레이아웃을 시각적으로 보여주는 맵. 구역별/층별 기업 부스 위치 표시.

---

### 2. 플랫폼 핵심 기능 및 목적

- **배경:** 매년 전국에서 수백 건의 채용박람회가 열리지만, 기존 채용 사이트(사람인, 잡코리아)는 채용 공고 중심이며, "채용 이벤트 자체를 관리"하는 통합 플랫폼은 부족. 특히 상담 예약, 이력서 사전 제출, 참가 기업 탐색을 온라인에서 원스톱으로 할 수 있는 서비스가 없음.
- **목적:** 채용박람회를 개설하고, 기업 부스 관리·구직자 참가 신청·1:1 상담 예약·이력서 제출을 통합 제공하여, 주최측·기업·구직자 3자를 연결하는 플랫폼 구축.
- **스토리 강점:** *"직접 구직 과정에서 채용박람회의 불편함을 느끼고, 이를 해결하기 위해 만들었습니다"* — 면접에서 강력한 동기 부여 스토리.
- **Cloud Native 연결:** 박람회 접수 기간에 트래픽 집중 → Docker 컨테이너 기반 배포. 상담 예약 동시성 제어 → Redis 분산 락으로 슬롯 충돌 방지.

---

### 3. 핵심 기술 포인트

- **상담 예약 시스템:**
  - 시간대별 슬롯(15분/30분 단위) 관리
  - 동시 예약 충돌 방지: DB 레벨 비관적 락(SELECT FOR UPDATE) 또는 Redis 분산 락
  - 예약 상태 관리: AVAILABLE → RESERVED → COMPLETED / CANCELLED
  - react-big-calendar로 캘린더 UI 구현 (드래그 앤 클릭으로 슬롯 선택)
- **이력서 파일 관리:**
  - Spring Boot MultipartFile로 파일 업로드
  - 파일 타입 검증(PDF/DOCX만 허용), 용량 제한(10MB), 파일명 UUID 변환
  - 접근 권한 관리: 이력서를 제출받은 기업만 열람 가능
  - 스토리지: 로컬 또는 AWS S3
- **역할 기반 접근 제어:**
  - SEEKER(구직자): 참가 신청, 이력서 제출, 상담 예약
  - COMPANY(기업): 부스 관리, 채용 공고, 이력서 열람
  - ORGANIZER(운영자): 이벤트 개설, 기업/참가자 관리
  - ADMIN(관리자): 전체 시스템 관리
