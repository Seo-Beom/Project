PROBLEM

왜 이 프로젝트를 하는가?

🔍

정보의 파편화

축제 정보가 지자체 홈페이지, 블로그, SNS 등에 흩어져 있어 한눈에 파악하기 어려움

🗺️

통합 플랫폼 부재

축제 일정·위치·후기를 하나의 플랫폼에서 통합 조회할 수 있는 서비스가 없음

💬

사용자 참여 콘텐츠 수요

후기·사진 공유 및 커뮤니티 소통 등 사용자 참여형 콘텐츠에 대한 수요가 존재

📊

공공데이터 활용 가능

공공데이터포털(data.go.kr)의 축제/행사 API를 활용하여 실제 데이터 기반 서비스 구현 가능

SOLUTION

프로젝트 목표

🗺️

지도 기반 통합 탐색

카카오맵 API 연동, 시/도별 클러스터링, GPS 기반 주변 축제 탐색

🔐

다중 권한 시스템

비회원(조회) + 회원(참여) + 관리자(운영) 3종 역할 기반 기능 제공

⭐

커뮤니티 생태계

지난 축제 후기·별점·베스트 포토 등 사용자 참여 콘텐츠 생태계 조성
