## 주요 인터랙션

| 요소 | 동작 |
|------|------|
| 이미지 갤러리 | 썸네일 클릭 시 메인 이미지 전환, 라이트박스 |
| 즐겨찾기 버튼 | 🔒 비회원 → 로그인 유도 / 회원 → 토글 (하트 채움/비움) |
| 후기 작성 버튼 | 🔒 비회원 → 로그인 유도 / 회원 → 작성 모달 오픈 |
| 길찾기 | 카카오맵 외부 링크 오픈 |
| 더보기 | 축제 소개 텍스트 펼치기/접기 |
