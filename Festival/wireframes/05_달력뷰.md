## 주요 인터랙션

| 요소 | 동작 |
|------|------|
| 날짜 셀 클릭 | 해당 날짜 축제 목록 하단에 표시 |
| 축제 바(기간 표시) | 호버 시 축제명 툴팁, 클릭 시 축제 상세 이동 |
| 월/주/일 전환 | FullCalendar 뷰 모드 변경 |
| ◀/▶ 화살표 | 이전/다음 월 이동 |
| 축제 개수 ≤ 4개 | 카드 형식 표시 |
| 축제 개수 ≥ 5개 | 리스트 형식 표시 |
| 카드/리스트 클릭 | → 축제 상세 페이지 이동 |
