## 주요 인터랙션

| 요소 | 동작 |
|------|------|
| 기간 선택 | 차트 데이터 기간별 갱신 |
| 지역별 분포 | 지도 or 히트맵 형태로 시/도별 축제 수 시각화 |
| 차트 호버 | 상세 수치 툴팁 표시 |
| 인기 축제 클릭 | → 해당 축제 상세 페이지 이동 |
| 사용 라이브러리 | Chart.js 또는 Recharts |
