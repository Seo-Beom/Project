## 주요 인터랙션

| 요소 | 동작 |
|------|------|
| 히어로 배너 | 자동 슬라이드 (5초 간격), 수동 네비게이션 |
| 축제 카드 클릭 | → 축제 상세 페이지 이동 |
| "지도에서 탐색하기" | → 전국 축제 지도 페이지 이동 |
| "달력 보기" | → 달력 뷰 페이지 이동 |
| 미니 지도 클릭 | → 전국 축제 지도 페이지 이동 |
| GNB 로그인 | → 로그인 페이지 이동 |
