## 주요 인터랙션

| 요소 | 동작 |
|------|------|
| 클러스터 마커 클릭 | 줌 인 → 하위 마커 표시 |
| 개별 마커 클릭 | 인포윈도우 팝업 (축제명, 위치, 기간, 별점) |
| 인포윈도우 "상세보기" | → 축제 상세 페이지 이동 |
| GPS 현위치 버튼 | 현재 위치로 지도 이동 + 주변 축제 표시 |
| 필터 변경 | 지도 마커 + 목록 실시간 갱신 |
| 목록 항목 호버/클릭 | 해당 마커 하이라이트 + 지도 이동 |
| 모바일 하단 목록 | 바텀시트(Bottom Sheet) 드래그로 열기/닫기 |
