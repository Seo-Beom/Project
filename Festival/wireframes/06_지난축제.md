## 주요 인터랙션

| 요소 | 동작 |
|------|------|
| 카드 클릭 | → 축제 상세 페이지 이동 (후기 탭 포커스) |
| 만족도바 | 별점 평균을 시각적 프로그레스바로 표시 |
| 정렬 옵션 | 최신순 / 별점순 / 후기많은순 |
| 지역 필터 | 시/도 선택 필터링 |
| 연도 필터 | 2024 / 2025 / 2026 연도별 필터링 |
| 비회원 | 후기 열람 가능, 작성 불가 (상세 페이지에서 로그인 유도) |
