## 주요 인터랙션

| 요소 | 동작 |
|------|------|
| 로그인 버튼 | 이메일/비밀번호 검증 → JWT 발급 → 메인 페이지 리다이렉트 |
| 회원가입 버튼 | 유효성 검사 → 계정 생성 → 로그인 페이지 이동 |
| 이메일 입력 | 실시간 형식 검증 |
| 닉네임 중복확인 | API 호출로 중복 여부 확인 |
| 비밀번호 | 강도 표시 (약/보통/강) |
| 비회원이 글 작성 등 시도 시 | 이 페이지로 리다이렉트 + "로그인이 필요합니다" 안내 |
