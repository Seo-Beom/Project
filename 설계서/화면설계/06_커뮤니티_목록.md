## 레이아웃 구성

| 영역 | 구성 요소 | 설명 |
|------|---------|------|
| GNB | "커뮤니티" 탭 활성 | 상단 네비게이션 |
| 카테고리 탭 | 전체 \| Q&A \| 축제꿀팁 \| 먹거리 | 말머리 필터 |
| 필터 버튼 | 우측 상단 | 정렬/기간 필터 |
| 검색바 | 검색어 입력 | 제목+내용 검색 |
| 게시판 테이블 | 번호 \| 제목 \| 작성자 \| 댓글수 \| 조회수 \| 작성일 | 행 리스트 |
| 페이지네이션 | < 1 2 3 4 5 > | 하단 중앙 |
| 글쓰기 버튼 | 우하단 플로팅 | 🔒 로그인 필요 |

---

## 테이블 컬럼

| 컬럼 | 데이터 | 정렬 |
|------|------|------|
| 번호 | `postId` | 중앙 |
| 제목 | `title` (최대 50자 말줄임) | 좌측 |
| 작성자 | `nickname` | 중앙 |
| 댓글수 | `commentCount` | 중앙 |
| 조회수 | `viewCount` | 중앙 |
| 작성일 | `createdAt` (YYYY.MM.DD) | 중앙 |

## 인터랙션

| 액션 | 동작 | 이동 화면 |
|------|------|---------|
| 게시글 행 클릭 | 상세 페이지 이동 | → 화면 7 |
| 카테고리 탭 | boardType 필터 변경 (QNA/TIP/FOOD) | 현재 화면 |
| 검색 | keyword 파라미터 검색 | 현재 화면 |
| 글쓰기 버튼 | 글 작성 폼 표시 | 신규 작성 페이지 |
