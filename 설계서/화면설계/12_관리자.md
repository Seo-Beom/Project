## 사이드바 메뉴별 콘텐츠

### 공공 축제 리스트 (API_ADM_0002)
| 기능 | 설명 |
|------|------|
| 목록 | 관리자용 축제 테이블 (source, sourceId, apiModifiedAt 포함) |
| 수동 갱신 | "동기화" 버튼 → API_ADM_0003 호출 (BAT-003) |
| 숨김 | 특정 축제 비공개 처리 |

### 신고 관리 (API_ADM_0005~0006)
| 기능 | 설명 |
|------|------|
| 목록 | 신고 테이블 (대상/사유/상태) |
| 처리 | 삭제/경고/기각 선택 |

### 공지 관리 (API_ADM_0007~0010)
| 기능 | 설명 |
|------|------|
| 목록 | 공지 테이블 (제목/등록일/팝업여부) |
| 등록/수정 | 제목, 내용, 팝업 여부 입력 폼 |
| 삭제 | 확인 팝업 → 삭제 |

## API 호출 (대시보드)

```
# 지역별 통계
GET /api/admin/statistics/regions

# 사용자 활동 통계
GET /api/admin/statistics/activity?period=WEEK
```
