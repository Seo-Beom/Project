## 레이아웃 구성

| 영역 | 구성 요소 | 설명 |
|------|---------|------|
| GNB | 로고 "이음" + 메뉴탭 + 로그인/회원가입 | 전역 네비게이션 |
| 탭 필터 | 전체 \| 진행중 \| 진행전 | 활성 탭 하이라이트 처리 |
| 검색 + 필터 | 검색 입력창 + 필터 드롭다운 버튼 | 지역/카테고리 필터링 |
| 카드 그리드 | 3열 × 3행 (총 9개) | 축제 카드 반복 렌더링 |
| 페이지네이션 | < 1 2 3 4 5 > | 페이지 이동 |
| 뷰 전환 | 리스트 목록 토글 | 카드뷰 ↔ 리스트뷰 전환 |

---

## 축제 카드 구성

| 요소 | 설명 |
|------|------|
| 이미지 | API `firstimage` 썸네일 (없으면 기본 이미지) |
| 축제명 | `title` 최대 2줄, 말줄임 처리 |
| 기간 | `eventstartdate` ~ `eventenddate` (YYYY.MM.DD) |
| 별점 | `avgRating` ★ 4.5 형식 |
| 상태 배지 | 진행중(초록) / 진행전(파랑) 색상 구분 |

---

## 인터랙션

| 액션 | 동작 | 이동 화면 |
|------|------|---------|
| 카드 클릭 | 축제 상세 페이지 이동 | → 화면 2 |
| 탭 전환 | `status` 파라미터 변경 (ONGOING/UPCOMING) | 현재 화면 새로고침 |
| 검색 입력 + 엔터 | `keyword` 파라미터로 검색 | → 화면 1 (결과) |
| 필터 드롭다운 | `areaCode`, `category` 파라미터 변경 | 현재 화면 새로고침 |
| 리스트 토글 | 카드 그리드 ↔ 테이블 리스트 전환 | 현재 화면 |
| 페이지 번호 클릭 | `page` 파라미터 변경 | 현재 화면 |

---

## API 호출

```
GET /api/festivals?page=1&size=9&status=ONGOING&sort=latest
```

| 파라미터 | 값 | 비고 |
|---------|---|------|
| page | 1 | 현재 페이지 |
| size | 9 | 3열 × 3행 |
| status | ONGOING | 탭 선택에 따라 변경 |
| areaCode | (선택) | 필터 드롭다운 |
| keyword | (선택) | 검색바 입력값 |
| sort | latest | 기본 정렬 |
