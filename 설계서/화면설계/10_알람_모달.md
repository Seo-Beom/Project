## 레이아웃 구성 (모달 오버레이)

| 영역 | 구성 요소 | 설명 |
|------|---------|------|
| 배경 | 딤드 처리 | 메인 페이지 위 반투명 오버레이 |
| 모달 (500×600px) | 알림 모달 팝업 | 중앙 배치, 흰색 배경, 둥근 모서리 |
| 헤더 | "알림" + X 닫기 | 설정/프로필 아이콘 |
| 공지 팝업 | 공지사항 배너 | 상단 박스형 공지 영역 |
| 알림 리스트 | 4가지 카테고리 알림 | 스크롤 가능 목록 |

## 알림 카테고리

| 아이콘 | 카테고리 | 내용 예시 |
|-------|---------|---------|
| 📝 글 | 게시글 반응 | 작성하신 "마케팅 전략" 글에 새로운 반응이 있습니다 |
| ⚠️ 댓글 | 댓글 답글 | "프로젝트" 댓글에 "마케팅팀장"님이 답글을 달았습니다 |
| 💬 문의 | 문의 답변 | "디자인 문의"에 대한 답변이 등록되었습니다 |
| ❓ 문의 | 새 문의 | "개발 일정"에 대한 새로운 문의가 등록되었습니다 |

## 인터랙션

| 액션 | 동작 |
|------|------|
| X 클릭 | 모달 닫기 |

| 설정 아이콘 클릭 | 알림 설정 페이지 이동 (마이페이지 내) |
