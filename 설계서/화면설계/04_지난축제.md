## 레이아웃 구성

| 영역 | 구성 요소 | 설명 |
|------|---------|------|
| GNB | "지난축제" 탭 활성 | 상단 네비게이션 |
| 검색 + 필터 | 검색 입력창 + ① 필터 버튼 | 지역/카테고리 필터 |

| 페이지네이션 | < 1 2 3 4 5 > | 하단 중앙 |
| 뷰 전환 | ⑤ 리스트 목록 토글 | 하단 우측 |

## 화면 1과의 차이점

| 항목 | 화면 1 (전국 축제) | 화면 4 (지난 축제) |
|------|-----------------|-----------------|
| 탭 필터 | 전체/진행중/진행전 | 없음 (종료 고정) |
| API 파라미터 | status 변동 | status=ENDED 고정 |
| 카드 배지 | 진행중(초록)/진행전(파랑) | "종료" 회색 배지 |
| 리뷰 작성 | ENDED만 가능 | 모든 카드에서 가능 |
