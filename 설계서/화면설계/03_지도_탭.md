## 레이아웃 구성

| 영역 | 구성 요소 | 설명 |
|------|---------|------|
| GNB | "지도" 탭 활성 | 상단 네비게이션 |
| 좌측 패널 (300px) | 지역 필터 + 상태 필터 + 축제 리스트 | 오버레이 사이드 패널 |
| 메인 지도 | 카카오맵 전체 화면 | 한반도 지도 |
| 클러스터 마커 | 서울 15, 경기 12 등 | 지역별 축제 수 표시 |
| 개별 마커 | 축제 위치 핀 | 줌인 시 표시 |
| GPS 버튼 | 내 위치 이동 | 우하단 |
| 줌 컨트롤 | +/- 버튼 | 우측 |
