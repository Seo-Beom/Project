## 레이아웃 구성 (2컬럼)

### 좌측 사이드바 (250px)

| 메뉴 항목 | 기능 | 비고 |
|---------|------|------|
| 사용자 프로필 | 아바타 + 닉네임 표시 | 상단 고정 |
| ⭐ 즐겨찾기 목록 (찜) | 북마크한 축제 카드 그리드 | 기본 선택 |
| 📝 내 게시글 목록 | 내가 작성한 게시글 리스트 | 수정/삭제 가능 |
| ⭐ 내 리뷰 목록 | 내가 작성한 리뷰 리스트 | 수정/삭제 가능 |
| 💬 내 댓글 목록 | 내가 작성한 댓글 리스트 | 답글에 대한 댓글 포함 |
| 🔔 내 알림 설정 | 알림 수신 ON/OFF | 카테고리별 설정 |
| 📩 내 문의 보기 | 문의 내역 리스트 | 답변 상태 표시 |
| ✏️ 프로필 수정 | 닉네임/이미지 변경 | 프로필 편집 폼 |
| 🚪 탈퇴 | 회원 탈퇴 | 확인 팝업 |

### 우측 콘텐츠 (나머지 너비)

선택된 메뉴에 따라 내용 변경. 기본은 **즐겨찾기 목록** (축제 카드 그리드)

## 디자인 참고사항

| 항목 | 설명 |
|------|------|
| 링크 이동 | 게시글/리뷰 항목 클릭 시 해당 페이지로 이동, 수정·삭제 가능 |
| 삭제 처리 | 삭제 시 팝업 확인 필수 + 댓글 숨김 처리 |
| 댓글 노출 | 내가 단 댓글이 타인 페이지에서도 간보이도록 처리 |

## API 호출

```
# 프로필 조회

# 프로필 수정
PUT /api/profile  (multipart: nickname, profileImage)

# 즐겨찾기 목록 (축제 카드)
GET /api/favorites?page=1&size=12
```
