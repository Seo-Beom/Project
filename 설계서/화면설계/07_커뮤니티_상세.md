## 레이아웃 구성 (중앙 정렬, max-width 900px)

| 영역 | 구성 요소 | 설명 |
|------|---------|------|
| 상단 우측 버튼 | 수정 \| 삭제 \| 신고 | 본인 글: 수정/삭제, 타인 글: 신고 |
| 게시글 헤더 | 제목 (대형 볼드) | `title` |
| 메타 정보 | 날짜 \| 작성일 \| 작성자 | `createdAt`, `nickname` |
| 태그 행 | 축제, 지역, 카테고리 | 말머리 선택 표시 |
| 말머리 | Q&A / 축제꿀팁 / 먹거리 | `boardType` |
| 본문 | 상세 내용 영역 | `content` + 이미지 |
| 댓글 섹션 | 댓글 리스트 | 아바타 + 닉네임 + 내용 + 날짜 |
| 댓글 액션 | 👍 👎 🚨 | 좋아요/싫어요/신고 |
| 댓글 입력 | 텍스트 입력 + "등록" 버튼 | 하단 |

## 인터랙션

| 액션 | 동작 | 조건 |
|------|------|------|
| 수정 버튼 | 글 편집 폼 | 🔒 본인만 |
| 삭제 버튼 | 삭제 확인 팝업 → 삭제 | 🔒 본인만 (댓글 CASCADE) |
| 신고 버튼 | 신고 사유 입력 팝업 | 🔒 로그인 + 타인 글 |
| 댓글 등록 | 댓글 추가 | 🔒 로그인 필수 |
| 댓글 삭제 | 댓글 제거 | 🔒 본인 댓글만 |

## API 호출

```
# 상세 조회
GET /api/community/posts/{postId}

# 댓글 작성
POST /api/community/posts/{postId}/comments  { "content": "..." }

# 댓글 삭제
DELETE /api/community/comments/{commentId}

# 신고
POST /api/reports  { "targetType": "POST", "targetId": 5, "reason": "..." }
```
