## 레이아웃 구성

| 영역 | 구성 요소 | 설명 |
|------|---------|------|
| GNB | "달력" 탭 활성 | 상단 네비게이션 |
| 월 네비게이션 | ← 2026. 3 → | 좌/우 화살표로 월 이동 |
| 캘린더 그리드 | 일~토 7열, 5~6행 | 날짜별 축제 개수 배지 표시 |
| 축제 카드 행 | 4개 카로셀 카드 | 날짜 클릭 시 하단에 표시 |
| 페이지네이션 | 21 22 23 ... | 하단 |

---

## 캘린더 셀 구성

| 요소 | 설명 |
|------|------|
| 날짜 숫자 | 1~31 |
| 축제 배지 | "2개", "3개" 등 타원형 배지 (축제 있는 날만) |
| 선택 표시 | 클릭된 날짜에 파란 테두리 |
| 오늘 표시 | 오늘 날짜에 원형 배경 |

---

## 인터랙션

| 액션 | 동작 | 설명 |
|------|------|------|
| 날짜 셀 클릭 | 해당 날짜 축제 카드 리스트 표시 | 캘린더 하단에 가로 카드 행 |
| 축제 카드 클릭 | 축제 상세 페이지 이동 | → 화면 2 |
| ← / → 화살표 | 이전/다음 월 이동 | year, month 파라미터 변경 |

---

## API 호출

```
GET /api/festivals/calendar?year=2026&month=3
```

**응답에서 날짜별 축제 개수를 계산하여 배지 렌더링**
