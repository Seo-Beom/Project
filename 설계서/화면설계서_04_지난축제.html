Description

Summary.

  • 종료된(ENDED) 축제를 카드형 리스트로 조회하는 화면
  • 화면 1(전국축제 메인)과 동일한 레이아웃 구조
  • 각 카드에 "종료" 배지 오버레이 표시
  • 종료된 축제에 대해 리뷰 작성 가능
1 GNB (상단 네비게이션)
로고 "이음" + 메뉴 탭: 전국축제 | 지난축제(활성) | 달력 | 커뮤니티 | 공지사항
"지난축제" 탭이 활성 상태 (볼드 하이라이트)
우측에 로그인 버튼 배치
2 검색바
"검색" 플레이스홀더 텍스트
축제명, 지역, 키워드로 종료된 축제 검색
Enter 또는 돋보기 아이콘 클릭으로 검색 실행
3 필터 + 정렬 드롭다운
"필터" 버튼 — 지역별/카테고리별 상세 필터
"필터 ∨" 드롭다운 — 정렬 옵션 (최신순, 인기순 등)
필터 변경 시 API 재호출하여 카드 목록 갱신
4 종료 배지 카드 그리드
3열 × 3행 레이아웃으로 축제 카드 배치
각 카드 구성:
· 축제 대표 이미지 (상단)
· "종료" 배지 오버레이 (우상단, 회색 배지)
· 축제 이름 (굵은 텍스트)
· 기간 (YYYY.MM.DD - YY.MM.DD 형식)
카드 클릭 → 축제 상세 페이지 이동 (동일 레이아웃)
5 페이지네이션 + 리스트 목록 토글
< 1 2 3 4 5 > 페이지 이동
우하단 "리스트 목록" 버튼 — 카드 그리드 ↔ 리스트 뷰 전환
서버사이드 페이징 (page, size 파라미터)

Check Point

· API 호출 시 status=ended 파라미터 고정
· 카드 클릭 시 축제 상세 페이지 (/festivals/{id})로 이동
· 종료된 축제 상세에서 리뷰 작성 가능 (ENDED 상태)
· 전국축제 메인과 동일한 검색/필터 UX 유지
· "종료" 배지는 카드 이미지 우상단에 오버레이
· 비회원도 목록 조회 가능
