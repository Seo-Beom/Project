Description

Summary.

  • 전국의 축제를 카드형 그리드로 조회하는 메인 화면
  • 탭(전체/진행중/진행전/지도)으로 상태별 필터링 가능
  • 검색 + 지역/카테고리 필터로 원하는 축제 빠른 탐색
  • 공공 API + 자체 등록 축제 데이터 통합 제공
1 GNB (상단 네비게이션)
로고 "이음" + 메뉴 탭: 전국축제 | 지난축제 | 달력 | 커뮤니티 | 공지사항
우측에 로그인/회원가입 버튼 배치
2 검색바 + 필터 버튼
축제명, 지역, 키워드로 축제 검색 가능
"필터" 버튼 클릭 시 지역별/카테고리별 상세 필터 모달 표시
3 탭 필터 (상태 전환)
· 전체 — 모든 축제 표시
· 진행중 (기본 활성, 밑줄 하이라이트) — status=ongoing
· 진행전 — status=upcoming
· 지도 — 지도 뷰 화면으로 전환
4 축제 카드 그리드
3열 × 3행 레이아웃으로 축제 카드 배치
각 카드 구성:
· 축제 대표 이미지 (상단)
· 축제명 (굵은 텍스트)
· ★ 평균 별점 (우측 상단)
· 기간 (2026.04.01 ~ 04.10)
카드 클릭 → 축제 상세 페이지 이동
5 별점 표시
각 축제 카드에 ★ 4.5 형태로 평균 별점 표시
API_REV_0020 리뷰 통계 데이터 연동
6 페이지네이션
< 1 2 3 4 5 > 형태의 페이지 이동
서버사이드 페이징 (page, size 파라미터)
7 리스트/그리드 뷰 토글
우하단 "리스트 목록" 토글 스위치
카드 그리드 ↔ 테이블 리스트 뷰 전환

Check Point

· 카드 클릭 시 축제 상세 페이지 (/festivals/{id})로 이동
· 탭 전환 시 status 쿼리 파라미터 변경하여 API 재호출
· 검색/필터 결과에 따라 카드 목록 동적 갱신
· 지도 탭 클릭 시 지도 뷰 화면으로 전환
· 비회원도 목록 조회 가능, 찜하기는 로그인 필요
